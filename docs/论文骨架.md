# 本科毕业设计论文骨架

题目：基于 Vue 3 与 Django 的学生睡眠管理系统设计与实现

## 摘要

说明青少年睡眠管理的现实背景、系统目标、技术路线和核心成果。突出学生自管、老师监督、家长关注三方协同。

## 第一章 绪论

1. 研究背景与意义
2. 国内外研究现状
3. 本课题研究内容
4. 论文组织结构

## 第二章 相关技术

1. Vue 3 与前端组件化开发
2. Django REST Framework
3. JWT 认证机制
4. ECharts 数据可视化
5. 基于规则引擎与大模型兜底的睡眠建议生成

## 第三章 系统需求分析

1. 角色需求：学生、老师、家长、管理员
2. 功能需求：打卡、统计、预警、消息、AI 建议、报表导出
3. 非功能需求：隐私保护、可维护性、可演示性
4. 业务规则：补卡 3 天、2 小时自改、凌晨跨天归属、周末阈值放宽

## 第四章 系统设计

1. 总体架构设计
2. 数据库设计
3. API 接口设计
4. 权限与隐私边界设计
5. 睡眠评分引擎设计

## 第五章 系统实现

1. 用户认证与邀请码绑定实现
2. 睡眠打卡与评分实现
3. 统计可视化与排行榜实现
4. 预警与消息通知实现
5. AI 建议与 Mock 兜底实现
6. 报表导出与演示数据实现

## 第六章 系统测试

1. 测试环境
2. 单元测试：评分引擎
3. 接口测试：认证、打卡、补卡、老师代改
4. 前端构建与浏览器联调
5. 测试结果分析

## 第七章 总结与展望

总结系统完成度、创新点和不足。展望移动端、小程序、更多班级、多维健康数据接入等方向。

## 参考文献

按学校格式补充睡眠健康、教育信息化、前后端分离开发等相关文献。
